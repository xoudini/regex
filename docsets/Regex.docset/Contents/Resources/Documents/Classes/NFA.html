  • init(with:matching:)

    Designated initializer. Creates an NFA with a single standard State, and an epsilon-transition from the initial state to the standard state, and a conditional transition from the standard state to the terminal state.

    Warning

    The given State can not be of initial or terminal type.

    Declaration

    Swift

    init(with state: State, matching symbol: Symbol)

    Parameters

    state

    The State to embed between the initial and terminal states.

    The Symbol to match in order to pass to the terminal state.
